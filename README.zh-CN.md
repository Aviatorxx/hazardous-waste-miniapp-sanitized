# 危废热处理微信小程序

[English README](./README.md)

这是一个用于公开展示的脱敏版本仓库，对应一个面向环境科学业务场景的前后端一体化微信小程序项目。

## 项目简介

项目由微信小程序前端和 Spring Boot 后端组成，主要用于危废信息管理、热力学特性展示以及配伍模拟分析。

## 仓库结构

- `frontend/`：微信小程序前端
- `backend/`：Spring Boot 后端服务

## 主要功能

- 危险废物目录查询与详情展示
- 理化特性查询与分类信息展示
- 热力学特性浏览，支持光谱图与图片预览
- 配伍检查与配伍模拟计算流程
- 提供检索、上传、计算、结果获取等 REST API

## 技术栈

- 前端：微信小程序原生框架
- 后端：Java 11、Spring Boot、MyBatis-Plus、MySQL
- 接口文档：Springdoc OpenAPI

## 脱敏说明

- 本仓库是用于作品集展示的脱敏公开版，不是原始部署仓库。
- 已移除或替换真实服务器地址、数据库凭据、小程序密钥、上传文件和私有环境配置。
- 已排除编译产物、日志、IDE 配置和仅限本地开发使用的文件。

## 本地运行

1. 将后端配置替换为你自己的数据库和小程序环境参数。
2. 修改 `frontend/config/api-config.js`，指向你自己的后端服务地址。
3. 使用微信开发者工具导入 `frontend/`。
4. 在 `backend/` 中启动 Spring Boot 服务。
