# Hazardous Waste Thermal Treatment Mini Program

Sanitized showcase version of a full-stack WeChat Mini Program project developed for an environmental science use case.

## Structure

- `frontend/`: WeChat Mini Program client
- `backend/`: Spring Boot backend service
- `docs/algorithm-docs/`: matching and blending algorithm notes

## Features

- Hazardous waste directory query
- Physical property query
- Thermal property display and image browsing
- Compatibility checking and blending simulation
- REST APIs for search, calculation, upload, and result export

## Tech Stack

- Frontend: WeChat Mini Program native framework
- Backend: Java 11, Spring Boot, MyBatis-Plus, MySQL
- Docs: algorithm notes for blending and matching logic

## Notes

- This repository is a sanitized portfolio version.
- Real deployment addresses, credentials, uploaded files, and private environment configuration have been removed or replaced with placeholders.
- To run locally, update the backend and frontend configuration with your own environment values.
